#  Netflix SQL Data Analysis

##  Project Overview
This project analyzes a Netflix-like dataset using SQL to extract insights about users, subscriptions, and content performance.

##  Database Schema
- Users  
- Movies  
- Subscriptions  
- WatchHistory  

##  Key Questions
- Total number of users?  
- Which country has the most users?  
- Which subscription plan generates the most revenue?  
- Most watched movies?  
- Most popular genres?  
- Average watch time per user?  

##  Key Insights
- Premium plan generates highest revenue  
- Movies dominate user consumption  
- Sci-Fi & Horror genres are popular  
- Some countries show higher engagement  

##  Skills Used
- SQL Joins  
- Aggregations  
- Window Functions  

##  Tools
- MySQL
